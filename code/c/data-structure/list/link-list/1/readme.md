
## 单链表运算演示

## 运行
```bash
# 插入运算
gcc ./insert.c -o ./temp/bin/insert && ./temp/bin/insert
# 删除运算
gcc ./delete.c -o ./temp/bin/delete && ./temp/bin/delete
# 定位运算
gcc ./locate.c -o ./temp/bin/locate && ./temp/bin/locate
# 读取元素
gcc ./get.c -o ./temp/bin/get && ./temp/bin/get
# 创建单链表
gcc ./create.c -o ./temp/bin/create && ./temp/bin/create
# 删除重复结点
gcc ./remove-repeat.c -o ./temp/bin/remove-repeat && ./temp/bin/remove-repeat
```
